nginx学习

  • nginx
web服务器

1. 什么是nginx

一个服务器软件，apache/tomcat/iis

nginx是一款高性能的服务器软件
可以发布网站实现负载均衡，反向代理服务器
也可以是一个代理邮件服务器
负载均衡：为了减小服务器压力，需要将用户访问信息分别引像服务器集群，减小服务器压力，提高响应效率，避免服务器崩溃

  • iis:只能部署在window上，不如linux做服务器好
  • tomcat：面向java重量级
  • apache：稳定、开源、跨平台，不支持高并发
  • nginx：轻量，高性能、高并发，内存消耗少，成本低，rewrite功能不够强大，模块没有apache多

2. nginx服务器搭建

linux系统中搭建nginx服务器

  1. 下载源码编译nginx-1.8.0.tar.gz tar -zxvf nginx-1.8.0.tar.gz
  2. cd nginx-1.8.0&&./config(保证有gcc gcc-c++)
  3. make(pre/lib模块安装)
  4. make install

3. 启动、停止、重载配置

  1. nginx -c path/to/nginx.conf
  2. nginx -s stop或者nginx -s quit或者pkill -9 nginx
  3. nginx -s reload
  4. 检查配置文件是否正确 nginx -t
